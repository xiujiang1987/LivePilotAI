## 🔧 系統需求

### 基本需求
- **Python**: 3.8+
- **作業系統**: Windows 10/11
- **記憶體**: 4GB+ 推薦
- **硬碟空間**: 2GB+

### 可選需求 (完整功能)
- **OBS Studio**: 28.0+ (直播功能)
- **網路攝影機**: HD 1080p+ (情緒偵測)
- **麥克風**: 任何 USB 麥克風 (語音互動)
