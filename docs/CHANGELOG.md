# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-06

### Added
- Initial stable release.
- Camera Management Module with real-time capture support.
- API Documentation in `docs/API.md`.
- Basic Unit Tests in `tests/unit/`.
- Pre-commit hooks for code quality (black, isort, flake8).
- `pyproject.toml` configuration for development tools.

### Fixed
- Standardized test file naming convention (`test_*.py`).
- Moved legacy script-style tests to `scripts/`.
- Fixed dependency installation issues (removed dlib).

### Changed
- Refactored `requirements.txt` to use modern package versions.
- Improved project structure for better maintainability.
