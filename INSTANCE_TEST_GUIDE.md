## 🔧 如果遇到問題

### 常見問題解決：

#### 1. 攝像頭無法啟動
```powershell
# 檢查攝像頭
python -c "import cv2; cap = cv2.VideoCapture(0); print('攝像頭可用' if cap.isOpened() else '攝像頭不可用'); cap.release()"
```

#### 2. 模組導入失敗
```powershell  
# 檢查依賴
python -c "import cv2, numpy as np, tensorflow as tf; print('所有依賴正常')"
```

#### 3. 性能問題
- 降低解析度: 640x480 → 320x240
- 降低幀率: 30 FPS → 15 FPS
- 關閉其他應用程式釋放資源
