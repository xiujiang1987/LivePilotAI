## 🚀 新的使用方式

### 基本啟動命令
```powershell
# 🎯 語意化主要啟動方式
python main.py

# 測試模式
python main.py --mode=test

# 示範模式
python main.py --mode=demo
```

### 分類執行命令
```powershell

python tests/performance_benchmark.py  # 效能測試
```
